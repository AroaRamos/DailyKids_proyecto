# DailyKids · Panel de Educadora

**DailyKids** es una plataforma educativa centrada en mejorarla seguridad en la comunicación, el seguimiento y la gestión diaria en escuelas infantiles.  

## Funcionalidades en desarrollo

- Lógica JS para editar perfil (guardar email, contraseña, imagen)
- Pestañas en los modales (Registro diario / Evaluaciones)
- Mostrar iconos pequeños junto a cada niño indicando si ya se ha rellenado la ficha del día
- Desactivar el formulario de registro diario si se marca ausencia
- Mostrar badges (números) en los botones:
  - Nuevos mensajes
  - Eventos del día
  - Turnos pendientes

## Funcionalidades futuras

### Vista educadora
- Página **"Mi Aula"** con los niños en cajas
  - Modal individual al hacer clic
  - Campos:
    - Comida: desayuno / primer plato / segundo / postre
    - Cacas (selector numérico)
    - Checklist: toallitas, pañales
    - Observaciones
    - Registro de faltas
    - Evaluaciones individuales
- Página **Calendario**
  - Eventos generales, cumpleaños, reuniones
  - Posibilidad de consulta por días o meses anteriores
- Página **Turnos**
  - Horarios diarios, personal de apoyo, cambios

### Vista administradora
- Página exclusiva con funcionalidades:
  - Crear aulas
  - Añadir o eliminar niños
  - Gestionar educadoras y turnos
  - Exportar reportes (PDF, CSV)

### Vista familias
- Ver el resumen diario del niño
- Notificaciones importantes
- Historial de asistencia y evaluaciones
- Comunicación con educadora vía mensajes
